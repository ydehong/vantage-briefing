Setup guide
1
Create a bot
Open Telegram → search @BotFather → send /newbot → follow prompts → copy the token it gives you.
2
Get your Chat ID
Search @userinfobot in Telegram → start it → it replies with your numeric Chat ID. Paste it above.
3
Start your bot
Search for your bot's username in Telegram → tap Start. This is required before it can message you.
4
Test and activate
Click "Send test message" above. Then go to Live Intelligence → Start monitoring. Alerts will fire automatically with each intelligence refresh.
iOS Shortcuts — runs on a schedule, no server needed
1
Open Shortcuts app on your iPhone
Tap + (new shortcut) → tap Add Action → search for "Get contents of URL".
2
Configure the API call
Set URL to http://localhost:3131, Method to POST, Headers and Body as shown below.
Headers
x-api-key: YOUR_ANTHROPIC_KEY Content-Type: application/json anthropic-version: 2023-06-01
Body (JSON)
{ "model": "claude-sonnet-4-6", "max_tokens": 400, "messages": [{ "role": "user", "content": "You are a value investor's market monitor. In under 300 words give me: (1) one macro signal relevant to US and Singapore value stocks today, (2) one actionable entry or exit trigger for a value investor, (3) one risk to watch this week. Be direct, no preamble." }] }
3
Parse the response
Add action: "Get Dictionary Value" → key: content → then "Get Item from List" → first item → "Get Dictionary Value" → key: text.
4
Send a notification
Add action: "Show Notification" → title: "Vantage Intel" → body: the text from step 3.
5
Schedule it
Go to the Automation tab → + → Time of Day → set to 8:30am daily (or Mon/Wed/Fri for less noise) → select your shortcut → turn off "Ask Before Running".
Make.com — cloud scheduler, works even when browser is closed
1
Create a free Make.com account
Go to make.com → sign up free (1,000 operations/month included) → create a new Scenario.
2
Add a Schedule trigger
First module: Schedule → set to "Every day at 08:30" (SGT = UTC+8, so set UTC 00:30).
3
Add HTTP module → Make a request
URL: http://localhost:3131 · Method: POST
Headers to add
x-api-key → YOUR_ANTHROPIC_KEY Content-Type → application/json anthropic-version → 2023-06-01
Body (raw JSON)
{"model":"claude-sonnet-4-6","max_tokens":400,"messages":[{"role":"user","content":"Value investor market monitor. US and SGX/SEA focus. Under 250 words: (1) one macro signal today, (2) one entry or exit trigger for value stocks, (3) one risk to watch. Direct, no preamble."}]}
4
Parse + send to Telegram
Add Telegram Bot module → "Send a Message" → Map: Chat ID (your ID) → Text: {{3.content[].text}} (adjust path from HTTP response).
5
Activate the scenario
Toggle the scenario ON. It will now call the Anthropic API every morning and push the summary to your Telegram.

What triggers an update alert
AI Model New Claude model release (claude-opus-*, claude-sonnet-*) — checked against latest known version. New models often produce substantially better financial analysis.
Macro Regime Rate cycle transitions, recession signals, or major central bank pivot — prompts your long-term screening criteria to be re-run with updated macro context.
Prompt Strategy Quarterly self-evaluation — the app asks Claude to critique its own analysis framework and flag any improvements. Results are logged here.
